# project_hincelin_dwa

## Installation
Using Git Bash

### folder creation
<pre><code>mkdir DWA
cd DWA
git init
git pull https://github.com/nicotitine/Projet_DWA</code></pre>

### launch client
<pre><code>cd client
./gradlew
./gradlew appStart</code></pre>

### launch server
<pre><code>cd server
./gradlew
./gradlew bootRun</pre></code>
